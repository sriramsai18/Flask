About the Performance Prediction App

A lightweight Flask web app that predicts employee productivity using a trained XGBoost model and real production input data.

Predictive Insight

The app evaluates employee productivity based on factors like department, team, target, idle time and output metrics.

Data-Driven Decisions

Managers can use the prediction results to identify coaching needs, allocate resources, and improve production planning.

Built with Flask

Clean Flask routing and templates provide a simple interface to collect input and display model results instantly.

How it works

Enter employee and production details, then submit the form. The app forwards the values to the XGBoost model and returns a productivity category.

  • Quarter, day, month, team and department inputs
  • Work metrics such as SMV, overtime and incentive
  • Idle time, idle men, workers and style changes
  • Model prediction shown on a clean results page
Why this app helps

It helps identify productivity patterns early, reduce waste, and support better decision-making in manufacturing environments.

Use cases:

  • Team performance review
  • Resource allocation
  • Process improvement
